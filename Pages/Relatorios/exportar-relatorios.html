COMO EXPORTAR RELATÓRIOS?

A exportação oficial garante que os dados consolidados sejam formatados em documentos prontos para arquivamento e apresentações institucionais.

Formatos Disponíveis

Após a conferência dos dados em tela, o EduAloca permite a extração das tabelas em formatos versáteis. O formato PDF é o padrão recomendado para impressão e arquivamento oficial, sendo gerado automaticamente com o layout institucional da instituição, garantindo seriedade e padronização visual em todos os documentos pedagógicos.

Fluxo de Exportação

Para finalizar seu relatório, basta clicar no botão Exportar localizado no topo da pré-visualização tabular. O sistema então processará a renderização do layout e iniciará o download automático. Esse processo é otimizado para que grandes volumes de dados de alocação sejam convertidos sem perda de performance, mantendo a fidelidade das informações consultadas no PostgreSQL.
