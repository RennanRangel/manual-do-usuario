Como cadastrar os dados?

O fluxo de cadastro í© o processo fundamental para alimentar inteligência de alocaçío do sistema.

O que este artigo aborda:

  • Preenchimento de formulários estruturados
  • Regras de validaçío em tempo real
  • Confirmando e salvando na base

Preenchimento Estruturado

Para iniciar um novo registro, seja de uma Unidade Curricular, Turma ou Ambiente, acesse o botío "Novo Cadastro" no topo da página de gerenciamento. O EduAloca utiliza formulários inteligentes que se adaptam ao tipo de dado inserido, garantindo que nenhum campo obrigatório seja esquecido durante a inserçío.

Ao cadastrar uma turma, por exemplo, o sistema solicitará o curso vinculado, o semestre atual e o período (matutino, vespertino ou noturno). Estas chaves primárias sío essenciais para que o algoritmo de alocaçío funcione corretamente nas etapas posteriores.

Validaçío em Tempo Real

Diferente de sistemas legados, o EduAloca possui uma camada de validaçío ativa. Se você tentar cadastrar um ambiente (sala) com uma capacidade inferior ao número de alunos previstos para aquela turma, o sistema impedirá o salvamento de um conflito logístico futuro.

Confirmaçío de Dados

Após preencher todos os campos e passar pelas validações automáticas, clique em "Confirmar". Este comando nío apenas armazena o dado localmente, mas tambí©m dispara uma rotina de atualizaçío na matriz geral, permitindo que a coordenaçío visualize o novo elemento instantaneamente no dashboard de ocupaçío.
