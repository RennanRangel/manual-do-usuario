Como cadastrar os dados?

Última atualização 29 de abril de 2026

O fluxo de cadastro é o processo fundamental para alimentar o sistema. É através dele que o EduAloca entende quais são as turmas, salas e professores disponíveis para organizar as alocações da melhor forma.

Preenchimento do Formulário

Para iniciar um novo registro, clique no botão "Novo Cadastro" na tela de gerenciamento. O sistema abrirá um formulário que solicita apenas as informações necessárias para aquele tipo de dado (como nome da turma, curso ou capacidade da sala).

Campos marcados com asterisco são obrigatórios. Certifique-se de preenchê-los com atenção para garantir que o sistema consiga realizar os cálculos de alocação corretamente posteriormente.

Validação Automática

O EduAloca auxilia você a evitar erros comuns. Por exemplo, se você tentar cadastrar uma sala com capacidade para 20 alunos em uma turma que possui 30, o sistema alertará sobre a inconsistência. Isso garante que a organização das aulas seja sempre viável e confortável para todos.

Salvando as Informações

Após conferir os dados, clique em "Confirmar". As informações serão salvas imediatamente e já ficarão disponíveis para consulta nos painéis de controle e relatórios. Caso precise alterar algo depois, você poderá usar a função de edição a qualquer momento.
